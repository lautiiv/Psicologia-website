# Psychology Website

Aplicación web desarrollada con Angular cuyo objetivo es ofrecer una presencia profesional para una psicóloga y, en el futuro, evolucionar hacia una plataforma integral de gestión de pacientes.

## Objetivos actuales

* Presentar información profesional.
* Mostrar los servicios ofrecidos.
* Facilitar el contacto con pacientes.
* Permitir la solicitud de turnos.
* Practicar el desarrollo de aplicaciones SPA utilizando Angular.

## Roadmap del proyecto

Este proyecto crecerá de manera incremental a medida que incorpore nuevos conocimientos en desarrollo web.

Las funcionalidades planificadas incluyen:

* Sistema de autenticación para la profesional.
* Panel administrativo privado.
* Gestión de pacientes.
* Historial de sesiones.
* Registro de notas clínicas por paciente.
* Administración de turnos.
* Seguimiento de la asistencia a las sesiones.
* Registro de pagos y control financiero.
* Estadísticas mensuales sobre pacientes, sesiones e ingresos.
* Integración con un backend y base de datos.

## Tecnologías

* Angular
* TypeScript
* HTML
* CSS
* Bootstrap

### Tecnologías previstas

* Django
* Django REST Framework
* PostgreSQL
* JWT Authentication
